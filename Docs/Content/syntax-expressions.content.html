Expressions

Expressions are combinations of values, variables, operators, and function calls that evaluate to a single value. They are the building blocks of statements and can be used wherever a value is expected. In Nova Script, expressions can be categorized into several types, including:

  • Literal Expressions
  • Variable Expressions
  • Operation Expressions
  • Compound Operation Expressions
  • Assignment Expressions
  • Function Call Expressions
  • Ternary Expressions
  • Dot Access Expressions
  • Array Access Expressions
  • Not Expressions
  • Type Check Expressions

Literal Expressions

The simplest form of expression. A literal is a fixed value written directly in the source code.

42          # NovaInt
3.14        # NovaFloat
"Hello"     # NovaString
true        # NovaBool
[1, 2, 3]   # NovaArray
null        # NovaNull

Variable Expressions

Refers to the value stored in a previously declared variable by using its identifier.

var score = 100
Print(score) # 'score' is the expression

Operation Expressions

Uses standard mathematical or logical operators to combine two values.

10 + 5
x * y
isValid && hasKey

Compound Operation Expressions

Performs an operation and an assignment in a single step.

x += 5  # Equivalent to x = x + 5
y *= 2  # Equivalent to y = y * 2

Assignment Expressions

Assigns a new value to a variable. In Nova Script, the assignment itself evaluates to the assigned value.

x = 20

Function Call Expressions

Invokes a function and evaluates to that function's return value.

math.Sin(1.5)
add(10, 20)

Ternary Expressions

A shorthand for if-else that evaluates to one of two values based on a condition.

Ternary operations must be wrapped in parentheses

var status = ("Adult" if (age >= 18) else "Minor")

Dot Access Expressions

Used to access properties or methods of a NovaValue.

player.name
io.Print("Hello")

Array Access Expressions

Retrieves a value from a NovaArray using an index inside square brackets.

var firstItem = inventory[0]

Not Expressions

Inverts a boolean value using the ! operator.

!true   # evaluates to false
!isDead # evaluates to true if isDead is false

Also inverts a NovaInt or NovaFloat using the - operator

Type Check Expressions

Checks if a value belongs to a specific type. returns a NovaBool.

if (x is "Int") { ... }

Note: built in types will always be capitalized and grouped as one word ie: Int, Float, String, Bool, Array, Null
