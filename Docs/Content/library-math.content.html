Math Library

The Math Library provides a collection of mathematical constants and functions for performing arithmetic, trigonometric, and randomization operations.

Constants

  • PI: The ratio of a circle's circumference to its diameter (~3.14159)
  • E: Euler's number, the base of natural logarithms (~2.71828)

Functions

Trigonometry & Basic

  • Sin(rad), Cos(rad), Tan(rad): Returns trigonometric results for radians
  • Sqrt(v): Returns the square root of a value
  • Abs(v): Returns the absolute value
  • Degrees(rad) / Radians(deg): Converts between angle units

Rounding & Comparison

  • Round(v), Floor(v), Ceil(v): Standard rounding functions
  • Min(a, b) / Max(a, b): Returns the smaller or larger of two values
  • Clamp(v, lo, hi): Constrains a value between a minimum and maximum
  • Lerp(a, b, t): Linearly interpolates between a and b by factor t

Randomization

  • RandF(): Returns a random float between 0.0 and 1.0
  • RandI(): Returns a random integer
  • RandFRange(min, max): Returns a random float within a range
  • RandIRange(min, max): Returns a random integer within a range
