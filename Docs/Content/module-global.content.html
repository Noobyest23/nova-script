Global Module

The Global Module provides functions that are automatically included in every NovaScript program.

Functions

  • Range(from : Number, to : Number, step : Number): Creates a new range of numbers
  • Execute(filepath : String): Executes a script file
  • Signal(): Creates a new signal see Types for member variables of signal
  • Int(value): Converts a value to an integer if possible
  • Float(value): Converts a value to a float if possible
  • String(value): Converts a value to a string (always possible)
  • Bool(value): Converts a value to a boolean if possible
  • Array(value, value, value): Converts a multiple values to an array
