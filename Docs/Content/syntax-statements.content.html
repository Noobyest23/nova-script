Statements

Statements are the building blocks of a program. They are the instructions that tell the computer what to do. In Nova Script, statements can be categorized into several types, including:

  • Variable Declarations
  • If Statements
  • While Loops
  • For Loops
  • Function Definitions
  • Return Statements
  • Type Declarations
  • Include Statements
  • Breakpoint
  • Super Breakpoint
  • Expression Statement

Variable Declarations

Variables are used to store data. In Nova Script, variables are typically declared using the var keyword.

var x = 10
var name = "Nova"

If Statements

If statements allow you to execute code based on a condition. Optional else blocks provide an alternative path.

if (x > 5) {
    Print("Greater")
} else {
    Print("Lesser")
}

While Loops

A while loop repeats a block of code as long as a specified condition remains true.

var i = 0
while (i < 5) {
    Print(i)
    i = i + 1
}

For Loops

For loops are used to iterate through a container

for (i in [1, 2, 3, 4, 5]) {
    Print(i)
}

Function Definitions

Functions are reusable blocks of code. They can take parameters and return values.

func add(a, b) {
    return a + b
}

Return Statements

The return statement exits a function and optionally sends a value back to the caller.

Type Declarations

Used to define new types or classes in Nova Script with an optional constructor

type Person {
    Person(name, age) {
        this.name = name
        this.age = age
    }
    var name
    var age
}

var bob = Person("Bob", 30)

Include Statements

Used to import external libraries or other Nova Script files or C++ modules into the current context.

include "math" as math
include "my_script.ns"

Breakpoint

Decompiles the current statement and prints the output, as well as printing all the variables in the scope and their values

Super Breakpoint

Decompiles the entire script and prints the output, as well as printing all the variables in the scope and their values

Expression Statement

An expression used as a standalone statement, such as a function call or an assignment.

Print("Hello World"); # A function call expression
