IO Library

The IO Library provides functions for input and output operations in Nova Script. It allows you to read from and write to files, as well as interact with the console.

Functions

  • Print(v1, v2...): Prints one or more values to the console
  • PrintWarning(v1, v2...): Prints one or more warning messages to the console
  • PrintError(v1, v2...): Prints one or more error messages to the console
  • ReadFile(path : String): Reads the contents of a file and returns it as a string
  • WriteFile(path : String, content : String): Writes a string to a file
  • Input(prompt : String): Displays a prompt and returns a String input from the user

Print, PrintWarning, PrintError, and Input only work with the console
