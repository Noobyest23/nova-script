Keywords

Keywords are reserved words in Nova Script that have special meaning and cannot be used as identifiers (like variable names, function names, etc.). They are fundamental to the syntax and structure of the language.

Here is a list of all keywords in Nova Script:

  • var Used to declare a new variable that can optionaly be initialized var my_variable = 10
  • const Used to declare a constant variable that cannot be reassigned. const myConstant = 10;
  • if Used to create a conditional statement. if (condition) { ... }, condition must evaluate to a NovaBool
  • else Used to create an alternative branch for a conditional statement. else { ... }
  • while Used to create a loop that continues while a condition is true. while (condition) { ... }
  • for Used to create a loop that iterates through a container. for (element in my_array) { ... }, in is another keyword and can be replaced with :
  • func Used to declare a new function. func myFunction() { ... }
  • return Used to return a value from a function. return value;
  • not Used to negate a boolean expression. not (condition) must evaluate to a NovaBool
  • and Used to perform a logical AND operation. condition1 and condition2 must evaluate to a NovaBool
  • or Used to perform a logical OR operation. condition1 or condition2 must evaluate to a NovaBool
  • type Used to declare a new type type MyType { ... }
  • include Used to include external files or modules
  • true Represents the boolean value true
  • false Represents the boolean value false
  • null Represents the absence of a value
