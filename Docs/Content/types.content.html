Types

Types are the different data types that Nova Script supports. They define the kind of data that can be stored and manipulated in variables, as well as the operations that can be performed on that data.

Here is a list of all built-in types in Nova Script:

  • NovaValue: The base type for all values in Nova Script. All other types inherit from NovaValue.
  • NovaInt: Represents integer numbers (e.g., 42, -7)
  • NovaFloat: Represents floating-point numbers (e.g., 3.14, -0.001)
  • NovaString: Represents sequences of characters (e.g., "Hello, World!")
  • NovaBool: Represents boolean values (true or false)
  • NovaArray: Represents ordered collections of values (e.g., [1, 2, 3])
  • NovaObject: Represents key-value pairs (e.g., { name: "Alice", age: 30 })
  • NovaSignal: Represents a signal that can be emitted and listened to

NovaValue

NovaValue is the base type for all values in Nova Script. All other types inherit from NovaValue. It defines the common interface and behavior for all values, such as how they are copied, assigned, and how they perform operations.

Methods

  • AddRef(): Increments the reference count for the value
  • Release(): Decrements the reference count for the value, if the value is reaches zero, calls Release() on all accessables and calls OnDestroy()
  • Copy(): Creates a copy of the value
  • Type(): Returns the type of the value as a string
  • ToString(): Returns a string representation of the value
  • Assign(NovaValue* rhs): Assigns a new value to this variable if possible
  • PerformOp(NovaValue* rhs, const NovaOperator& op): Performs an operation with another value
  • PerformCompoundOp(NovaValue* rhs, const NovaOperator& op): Performs a compound operation with another value
  • Access(const std::string& name): Returns the value of the member with the given name, or null if it does not exist
  • OnDestroy(): Called when the value is destroyed

NovaInt

Methods

  • Num(): Returns a pointer to the C++ integer
  • CNum(): Returns a const reference to the C++ integer

Operators

NovaInt supports the following operators with NovaInt and NovaFloat values:

  • +: Addition
  • -: Subtraction
  • *: Multiplication
  • /: Division
  • ==: Equality comparison
  • !=: Inequality comparison
  • <: Less than comparison
  • >: Greater than comparison
  • <=: Less than or equal to comparison
  • >=: Greater than or equal to comparison
  • +=: Addition assignment
  • -=: Subtraction assignment
  • *=: Multiplication assignment
  • /=: Division assignment

NovaString

Methods

  • Str(): Returns a pointer to the C++ string
  • CStr(): Returns a const reference to the C++ string

Operators

NovaString supports the following operators with NovaString values:

  • +: Concatenation
  • ==: Equality comparison
  • !=: Inequality comparison
  • +=: Concatenation assignment

Members

  • Length(): Returns the length of the string as a NovaInt
  • IsEmpty(): Returns true if the string is empty, false otherwise
  • StartsWith(str : String): Returns true if the string starts with the given prefix, false otherwise
  • EndsWith(str : String): Returns true if the string ends with the given suffix, false otherwise
  • Contains(str : String): Returns true if the string contains the given substring, false otherwise
  • ToUpper(): Returns a new string with all characters converted to uppercase
  • ToLower(): Returns a new string with all characters converted to lowercase
  • Trim(): Returns a new string with leading and trailing whitespace removed
  • Find(str : String): Returns the index of the first occurrence of the given substring, or -1 if not found
  • Replace(old : String, new : String): Returns a new string with all occurrences of the given substring replaced with another substring
  • Substr(start : Int, length : Int): Returns a new string that is a substring of the original string
  • Split(delimiter : String): Splits the string into an array of substrings using the given delimiter

NovaBool

Methods

  • B(): Returns a pointer to the C++ boolean
  • CB(): Returns a const reference to the C++ boolean

Operators

NovaBool supports the following operators with NovaBool values:

  • ==: Equality comparison
  • !=: Inequality comparison

NovaArray

Methods

  • Arr(): Returns a pointer to the C++ vector of NovaValue pointers
  • CArr(): Returns a const reference to the C++ vector of NovaValue pointers

Members

  • Size(): Returns the number of elements in the array as a NovaInt
  • PushBack(value : NovaValue): Adds a new element to the end of the array
  • Pop(): Removes the last element from the array
  • Clear(): Removes all elements from the array
  • Erase(index : Int): Removes the element at the specified index

NovaObject

A NovaObject is a value with no data other than accessables

Methods

  • PushBack(name : String, value : NovaValue): Adds a new member with the given name and value to the object

NovaSignal

A NovaSignal is a value that can be emitted and listened to. It allows for communication between different parts of a program, enabling more modular and maintainable code.

Members

  • Emit(args...): Emits the signal with the given arguments, calling all connected listeners with those arguments
  • Connect(listener : Function): Connects a listener function to the signal, which will be called whenever the signal is emitted
  • Disconnect(listener : Function): Disconnects a previously connected listener from the signal
  • GetConnections(): Returns a list of all connected listeners
