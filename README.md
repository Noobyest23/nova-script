Nova Script is an interpreted scripting language made to be integrated into other systems, made by a solo developer—powered by caffeine, late nights, and too much free time.
# What is Nova Script?

Nova Script is an in-development scripting language. Made with extendability and integration in mind.

The Documentation (just like the language itself) is also still in development. There may be pages missing for some things. Make sure to write a bug report for any pages that are missing.
